## Cloud AI Providers in Shinkai

Shinkai allows you to connect to powerful cloud-based AI models without any special hardware requirements. Just connect your API keys and start using state-of-the-art AI models on any device.

<Tip>
Cloud AI models are perfect for users without powerful hardware or those who need access to the latest, most capable AI models.
</Tip>

## Benefits of Cloud AI Models

- **No Hardware Requirements**: Run on any device, even with limited specs
- **Access to Cutting-Edge Models**: Use the latest models as soon as they're released
- **Specialized Capabilities**: Access models optimized for specific tasks
- **Quick Setup**: Start using AI immediately with just an API key

## Quick Guide: Adding a Cloud AI Provider

<Steps>
  <Step title="Get an API key">
    Sign up with your preferred provider and generate an API key.
    
    | Provider | API Key URL |
    |----------|-------------|
    | OpenAI | https://platform.openai.com/api-keys |
    | Claude | https://console.anthropic.com/settings/keys |
    | Gemini | https://aistudio.google.com/app/apikey |
    | Groq | https://console.groq.com/keys |
    | OpenRouter | https://openrouter.ai/settings/keys |
    | Together AI | https://api.together.ai/ |
  </Step>
  
  <Step title="Add the provider in Shinkai">
    In Shinkai > Go to `AIs` > `+ Add AI` > `+ Manually Add AI`
    1. Select your provider (e.g., OpenAI, Claude, etc.)
    2. Choose your preferred model
    3. Enter your API key
    4. Customize the AI Name (if desired)
    5. Click `Add AI`
    
    <Frame>
      <img src="/images/add-gpt4.jpg" style={{ borderRadius: '0.5rem' }} />
    </Frame>
  </Step>
</Steps>
