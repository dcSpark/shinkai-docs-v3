**Anthropic**'s Claude AI models are designed for natural language processing tasks and capable of handling multimodal inputs such as text, audio, and images.

<Tip>
For an overview of all cloud AI providers in Shinkai, see our [Cloud AI Models](/advanced/models/cloud-models) page.
</Tip>
  <Info> 
    You need to log in into [Anthropic](https://console.anthropic.com/dashboard) to generate your API keys.
     
  </Info>
  <Steps>
    <Step title="Generate your API key">
      Log in to [Anthropic's dashboard](https://console.anthropic.com/settings/keys) > Select Settings > API Keys.

      Click the `+ Create Key`button > Select a workspace > Give your key a name (e.g., Shinkai-Desktop)

      ![Claude](images/claude.jpg)

      <Icon icon="copy" iconType="duotone" /> Copy the key and go to your Shinkai app > Paste the key.
    </Step>
    <Step title="Select the model to install">
      When you select Claude in model provider, you will get the list of available models to install (e.g., 3.5 Sonnet, Haiku, Opus, etc.)
      
    </Step>
    <Step title="Complete the manual installation in your Shinkai app">
      
      In `AI Name` field you can customize how you want to see this AI displayed in your Chats. 
      <Note> This field only accepts alphanumeric characters and underscores. </Note>
      Click `Add AI`. You are all set! 

    </Step>
  </Steps>
