**Shinkai** is continuously pushing the boundaries of AI technology by offering experimental features that unlock new possibilities for users. 
These experimental tools are designed to **enhance user interaction with AI, improve data handling, and provide cutting-edge capabilities** for more personalized and efficient experiences. 


### How to enable Shinkai's experimental features

To access Shinkai's experimental features you must enable them. To do this simply go to `Settings` > `General` > Toggle `Enable Experimental Features` on.

You will see the features displayed on the left side menu.
