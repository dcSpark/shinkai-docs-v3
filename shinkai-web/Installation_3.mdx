---
title: "Installation & Setup Guide"
---

Good news: Shinkai Web does not require installation.

Because Shinkai Web runs fully in your browser, there is nothing to download, set up, or configure locally.  

You can access it instantly from any device — desktop, laptop, or tablet.

Requirements

Although no installation is needed, here are the recommended requirements for the best experience:

Supported browsers

- Chrome (recommended)

- Firefox

- Edge  

- Safari.

Internet connection

A stable connection is required for:

- cloud agents  

- tool-powered agents  

- x402 credit purchases  

Optional: Shinkai Desktop App

If you want to run **local models**, store **local files**, or work offline, you can complement the web version with the Shinkai Desktop App.

Log in and start using Shinkai Web

1. Go to the Shinkai Web link provided.  

2. Create an account or log in with your existing Shinkai account.  

3. Start using agents instantly — no setup needed.

Troubleshooting

If something isn't working:

- Refresh the page.  

- Ensure your browser is updated.  

- Try an incognito window to rule out extensions.  

If issues persist, visit the **Help & Support** section (coming soon) or contact the team.
