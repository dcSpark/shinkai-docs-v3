If you're new to Shinkai, start here. These steps will help you run your first agent on the browser in just a few minutes.

1. Create your account : 

Go to Shinkai Web (https://web.shinkai.com/) and sign up or log in with your existing Shinkai account.

2. Open the Explore Agents page  

Browse featured agents, starter templates, or your own saved agents.

3. Run your first agent  

Click any agent → “Chat” → and begin a conversation in the browser.

4. Try a tool-powered agent  

Select an agent that uses tools like web search, image generation, or text-to-video.  

This shows you how Shinkai Web goes beyond simple chat.

5. Personalize your prompts  

Save prompts you use frequently (e.g., “summarize in 5 bullets” or “rewrite more formal”) and attach them to an agent for consistent behavior.

6. Explore collaboration  

Try triggering a workflow that uses two or more agents together.Available directly from the web interface.

7. Top up AI credits (optional)  

If you want to run higher-volume or advanced agents, go to the x402 section and purchase additional AI credits.
